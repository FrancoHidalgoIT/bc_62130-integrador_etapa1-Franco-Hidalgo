# Para comezar a desarrollar

## Instalo las dependencias (leer el archivo package.json)

```sh
npm i
npm install
```

## Arracar el servidor de desarrollo

```sh
npm run dev
```


## Una vez terminado el desarrollo. Puedo buildear el proyecto.

```sh
npm run build
```
